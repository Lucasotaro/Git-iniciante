Aqui, nesta aba, reforçaremos os comandos dobre git, isto me ajudará
a fixar o conteúdo e pode lhe auxiliar no seu desenvolvimento pessoal.

Antes de tudo, é preciso inserir seus dados pessoais, como nome e email,
para isto, use os seguintes códigos: git config --global user.name "seu nome" | git config --global user.email "seu email"

Com o git instalado, crie uma pasta qualquer no seu dispositivo, nela use
o botão direito do seu mouse e abra o Git Bash dentro dela, a seguir voçê verá
os comandos e suas respectivas funções, frizando no básico dos comandos git:

git init: Cria um arquivo .git que fica oculto dentro da sua máquina, que contêm
alguns arquivos, no mais, inicia o seu processo;

touch index.html: Para criar o arquivo dentro da pasta desejada;

git add: Use para monitorar o seu processo (git add index.html);

git status: Nome bastante sugestivo, este comando faz extamente isto, mostra o estado do seu repositório;

git commit: Use os commits para consolidar suas ações ao seu repositório (git commit -m "comentário")

Para conferir este "comentário" uese git log, assim voçê poderá visualizar seu commit;

git rm --cached: É ultilizado para remover um arquivo desejado, basta usar o comando e informar o que deseja remover.
